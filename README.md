## 📖 Project Description

This project, “Journey to Mars,” is an immersive interactive storytelling website designed to simulate a cinematic space exploration experience. Instead of presenting static content, the website guides users through a narrative journey from Earth to Mars using smooth scrolling, animation, and interactive elements.

The core concept focuses on transforming a simple webpage into a dynamic experience. As users scroll, they transition through different stages of the journey—launch from Earth, travel through space, and arrival on Mars—creating a sense of progression and storytelling. GSAP and ScrollTrigger are used to implement scroll-based animations, including a moving rocket and section reveal effects, while Lenis enables smooth, fluid scrolling for a premium feel.

To enhance immersion, a dynamic starfield background with randomized motion and twinkling effects creates depth and realism. A zoom-based scaling effect is applied to simulate movement through space, adding a cinematic layer to the experience. Interactive elements such as clickable planet sections and subtle cursor-based rocket motion increase user engagement and make the interface feel responsive and alive.

The design follows modern UI principles, incorporating glassmorphism, gradient typography, and a dark space-themed color palette. The layout is fully responsive and optimized for performance across different devices.

Overall, this project demonstrates how animation, storytelling, and interactivity can be combined to create a compelling and memorable web experience.

## ✨ Features

* 🚀 Cinematic scroll-based storytelling
* 🌌 Dynamic starfield background with depth effect
* 🎬 GSAP animations (rocket movement, section transitions)
* 🧭 Smooth scrolling using Lenis
* 🖱️ Interactive elements (clickable planets, cursor interaction)
* 💎 Glassmorphism UI design
* 📱 Fully responsive (mobile, tablet, desktop)

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Animations:** GSAP + ScrollTrigger
* **Smooth Scroll:** Lenis
* **Deployment:** Vercel
